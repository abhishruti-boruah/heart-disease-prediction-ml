Machine learning model to predict heart disease using the UCI Heart Disease dataset.
# Heart Disease Prediction using Machine Learning

This project predicts the presence of heart disease using a machine learning model.

The model is trained on the UCI Heart Disease dataset which contains medical attributes such as age, cholesterol level, chest pain type, and heart rate.

## Technologies Used

Python
Pandas
Scikit-learn

## Dataset

UCI Heart Disease Dataset (Cleveland)

Features include:
- Age
- Sex
- Chest pain type
- Cholesterol
- Blood pressure
- Heart rate
- ECG results

Target:
1 = Heart Disease
0 = No Heart Disease

## Workflow

1. Load and preprocess the dataset
2. Split data into training and testing sets
3. Train a Random Forest classifier
4. Evaluate model accuracy
5. Use the trained model for prediction

## How to Run

Install dependencies

pip install -r requirements.txt

Train model

python train_model.py

Run prediction

python predict.py
